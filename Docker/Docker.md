# Docker 
is a tool for running application in isolated environments

## Container 
containers are an **abstraction** at app layer that package the **code** and the **dependencies** together. Multiple container can run in the same machine and share the **OS Kernel** with other containers, each running as isolated processes on user space.

## Docker Image 
Image is an template for creating an environment

a **Container** is run instance of image 

## Docker Volumes
Docker Volumes allows to share data between host and container and also between containers
